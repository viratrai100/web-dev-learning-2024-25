## 📁 Project Summaries

### 1. **Student Table**
An HTML table that displays student data in an organized tabular format.

### 2. **HTML Table Project - Rowspan and Colspan**
Demonstrates advanced table structuring using rowspan and colspan for merged cells.

### 3. **HTML Table Project - Rowspan and Colspan (Fixed Tags)**
A corrected version of Project 2 with `<thead>` wrapped properly in `<tr>` for semantic accuracy.

### 4. **HTML Table Project - Nested Rowspan Example**
Illustrates nested rowspan implementation for multi-level data hierarchy in a table.

### 5. **HTML Form Project - Basic Login Form**
A simple login form with input fields for username and password.

### 6. **HTML Form Project - Input Types Showcase**
Explores various input types like number, date, range, color, and more.

### 7. **HTML Form Project - Event Registration Form**
A complete registration form for an event with input validation and field organization.

### 8. **HTML Form Project - Job Application Form**
A job application layout including resume upload, dropdowns, and field grouping.

### 9. **HTML Form Project - Complex Form**
A detailed form with radio buttons, checkboxes, dropdowns, and various text inputs.

### 10. **HTML Project - Gaming Showcase Website**
A content-rich website with embedded YouTube videos, game descriptions, and media.

### 11. **Personal Portfolio / Resume Website**
Portfolio with personal details, skills, resume sections, project links, FAQs, and a contact form. Includes internal CSS for basic layout and image handling.
